Installation and Setup

This template is pure HTML5 and it doesn't require any special installation process. You can just upload it on a server or open it locally and it works.

Setup color scheme

To setup the color scheme (please see Colors) you need to change the CSS main file included in HTML files. To change the CSS main file, you have to change the file name at the top of every HTML files where the app.css is included. Please see below the file names that correspond to the five color schemes available for this template.

CSS

This template is based on Boostrap 3. For the bootstrap css and components, please check here

Apart from the CSS files used by the jquery plugins, this template uses one main CSS file (app.css) and four other CSS files that can replace the main one, if you want to change the color scheme (app-red.css, app-blue.css, app-magenta.css, app-yellow.css). Another CSS file is fullscreen-slider.css used in signin.html and signup.html files for the fullscreen slider and here you can change name files for images displayed in the slider:

.cb-slideshow li:nth-child(1) span { 
    background-image: url(../images/bg-1.jpg) 
}

HTML

Layout structure

The application is structured in: header, 1 sidebar for the main navigation and 2 main columns for the map and content.

As you can see in the HTML code bellow, the main layout components are given ids: header -> #header, sidebar navigation -> #leftSide, map -> #mapView, content -> #content

The homepage and blog pages have a different layout containing header, content and footer.

The header in wraped inside the hero of the page that contains the slideshow images (id: #hero-container). The content and footer have classes: content -> .home-content, footer -> .home-footer
